キー操作

Ctrlキーを使用するキー操作はCassava独特のものですが、 慣れると結構便利だと思います。

覚え方としては「１セルを１文字とみなす」です。

Ctrl + Ins セルの挿入。現在の内容は右に送られる。（Shiftを押していれば下向き）
Ctrl + Del セルの削除。右のセルを左に詰める。（Shiftを押していれば上向き）
Shift + Enter行の挿入。
Ctrl + Enter 行の挿入。カーソルから右のセルを挿入した行に移動。
Ctrl + BkSp 左のセルと連結。CSVファイルの「,」を消しているというイメージで。
Ctrl + Space カーソルのある列をすべて選択。
Shift + Spaceカーソルのある行をすべて選択。

このキー操作は、「オプション→キーカスタマイズ」で変更できます。
